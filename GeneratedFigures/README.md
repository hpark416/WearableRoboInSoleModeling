# Generated Figures

This folder stores generated visual outputs for the project.

## What to Place Here

- simulation result plots (e.g., force/displacement/time series),
- comparison charts across sole stiffness or other parameter sweeps,
- diagrams used in reports or presentations.

## Naming Guidance

Use descriptive names with context, for example:

- `grf_soft_vs_hard_sole.png`
- `metabolic_cost_vs_stiffness.png`
- `demographic_parameter_sweep_age_groups.png`
